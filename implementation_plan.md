# Парсинг PDF каталогов Pâté + визуальный интерфейс

## Что делаем

Два PDF каталога с разными наборами товаров. Нужно:
1. Спарсить **Старый каталог** полностью → база товаров
2. Спарсить **Новый каталог** → обновить изменившиеся цены, добавить новые товары
3. Показать в **красивом веб-интерфейсе**: что добавилось, что обновилось, что без изменений

---

## Структура PDF

### Паттерн одной страницы (3 товара на лист)
Каждая страница = секция категории (ТАРТЫ, ГАЛЕТЫ, СЛАДКИЕ ПИРОГИ, КИШИ, etc.)  
На странице **3 карточки товара**, расположенные вертикально (~y=80, ~y=255, ~y=425):

Для каждой карточки:
- **Название** (bold, sz=14) → y≈80/255/425, x≈21
- **Описание** (sz=10-11) → ниже названия
- **Размеры/вес** (sz=6) → мелкий текст
- **Цены** (sz=15, два числа) → x≈32 (мини), x≈113 (большой)
- **Подписи к ценам** (sz=9) → x≈34/107
- **NEW** метка (sz=14, x≈354) → если есть, товар новый в каталоге

### Категории
- Старый (34 стр): СЫТНЫЕ ПИРОГИ, КИШ, СЛАДКИЕ ПИРОГИ, СЕТЫ, ЧАЙНЫЕ НАБОРЫ, ХЛЕБ, ДЕСЕРТЫ, ТАРТЫ (внутри ДЕСЕРТОВ), СЭНДВИЧИ, ЗАВТРАКИ, НАПИТКИ
- Новый (33 стр): + ТАРТЫ отдельной секцией (стр 4-6) + ГАЛЕТЫ (стр 7), убраны СЛАДКИЕ ПИРОГИ 4-7

## Предлагаемые изменения

### [NEW] `parse_catalog.py`
Парсер PDF → JSON. Алгоритм:
1. Читаем все страницы обоих PDF через pymupdf
2. Определяем категорию страницы (заголовок CAPS, sz=24)
3. Группируем спаны по 3 зонам по Y (~y<170, ~170<y<340, ~y>340)
4. Для каждой зоны: bold-заголовок → имя, ближайшие цены → размеры, текст → описание
5. Вытаскиваем картинки из PDF и сохраняем в `images_new/`
6. **Merge логика**:
   - Старый каталог = база
   - Новый: совпало по имени → `status: "updated"` (если цены изменились) или `"unchanged"`
   - Новый: нет в старом → `status: "new"`
   - Старый: нет в новом → `status: "old_only"`

### [NEW] `catalog_merged.json`
```json
{
  "canonical": "Яблоко-фисташка тарт",
  "category": "ТАРТЫ",
  "description": "...",
  "sizes": {
    "мини": {"price": 1800, "old_price": 1800, "image": "yabloko-fistashka_tart.jpg"},
    "большой": {"price": 12000, "old_price": 8600, "image": "..."}
  },
  "status": "updated",
  "is_new_in_catalog": false
}
```

### [NEW] `catalog_viewer/index.html`
Визуальный SPA без фреймворков:
- Статистика сверху (Всего / Новые / Обновлённые / Без изменений / Только в старом)
- Фильтры по статусу и категории
- Карточки товаров: фото, название, категория, цены (было/стало), бейдж NEW/UPDATED
- Плавные анимации, тёмная тема, glassmorphism
